RAG

Retrieval-Augmented Generation Instead of using only a fine-tuned language model to generate the responses, a pre-trained language model combined with a document retriever can be used. This model, called a Retrieval-Augmented Generation model (RAG), first retrieves documents that are relevant to the input text from a provided dataset using a similarity search on the embeddings of the texts. Then it feeds these documents to the language model to generate a response which is relevant and properly worded.
RAG research paper

Model in this demo includes retriever models/vector stores for the Lenovo ThinkPad Gen 2 User Guide and the Epson L1210 Series Printer User Guide.
